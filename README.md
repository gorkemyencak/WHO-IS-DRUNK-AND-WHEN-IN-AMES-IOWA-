# WHO-IS-DRUNK-AND-WHEN-IN-AMES-IOWA-
Ames, Iowa, USA is the home of Iowa State University, a land grant university with over 36,000 students. By comparison, the city of Ames, Iowa, itself only has about 65,000 residents. As with any other college town, Ames has had its fair share of alcohol-related incidents. (For example, Google 'VEISHEA riots 2014'.) 

We will group, summarize, and visualize data on breath alcohol tests in Ames, Iowa, (home of Iowa State University) from 2013-2017. Some questions you will answer include, "What is the highest recorded value?" and "When do breath alcohol tests occur most?"
